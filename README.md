# Expense Budget Tracker Website

Marketing website for [Expense Budget Tracker](https://github.com/kirill-markin/expense-budget-tracker). Static Next.js site deployed on Vercel.

**Live:** [expense-budget-tracker.com](https://expense-budget-tracker.com/)

## Tech Stack

- Next.js 16 with App Router, TypeScript, React 19 Server Components
- Tailwind CSS v4 + CSS Modules
- Markdown processing: gray-matter, remark
- Deployed on Vercel

## Local dev

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Commands

- `npm run dev` — Development server with Turbopack
- `npm run build` — Production build
- `npm run lint` — ESLint linting
