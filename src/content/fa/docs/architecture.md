## نمای کلی سیستم

```
رابط مرورگر  -->  وب‌اپ Next.js  -->  Postgres (RLS)
                       |                ^
                       v                |
                 سرویس احراز هویت ------+
                       ^
                       |
          کلاینت‌های ماشینی از طریق API Gateway
                       ^
                       |
               Worker (FX fetchers) -----
```

پنج مؤلفه و یک پایگاه داده:

1. **web** — اپلیکیشن Next.js با داشبوردهای رابط کاربری و مسیرهای API
2. **auth** — ورود با Email OTP و راه‌اندازی اولیهٔ agent روی دامنهٔ auth
3. **sql-api** — AWS Lambda پشت API Gateway برای کلاینت‌های ماشینی
4. **worker** — دریافت روزانهٔ نرخ ارز از ECB، CBR و NBS
5. **Postgres** — تنها منبع حقیقت با RLS

## مدل داده

- **ledger_entries** — یک ردیف برای هر جابه‌جایی حساب، مانند درآمد، هزینه یا انتقال
- **budget_lines** — سلول‌های بودجه به‌صورت فقط‌الحاقی با قاعدهٔ آخرین نوشتن برنده است
- **budget_comments** — یادداشت‌های فقط‌الحاقی متصل به سلول‌های بودجه
- **workspace_settings** — ارز گزارش‌گیری و تنظیمات سطح فضای کاری
- **account_metadata** — فرادادهٔ هر حساب، مانند طبقه‌بندی نقدشوندگی
- **exchange_rates** — نرخ‌های روزانهٔ ارز برای تبدیل در زمان اجرای پرس‌وجو
- **workspaces** / **workspace_members** — ایزوله‌سازی چندمستاجره
- **accounts** — نمایی که از `ledger_entries` مشتق می‌شود

## چندارزی

همهٔ مبلغ‌ها با ارز بومی خودشان ذخیره می‌شوند. تبدیل به ارز گزارش‌گیری در زمان خواندن و از طریق اتصال‌های SQL با `exchange_rates` انجام می‌شود. هیچ پیش‌تبدیلِ اتلافی انجام نمی‌شود.

## احراز هویت

دو حالت از طریق متغیر محیطی `AUTH_MODE`:

- `none` — بدون احراز هویت، با یک فضای کاری محلی
- `cognito` — Email OTP بدون گذرواژه از طریق AWS Cognito، با ثبت‌نام باز

برای کلاینت‌های ماشینی، نقطهٔ ورودی عمومی برای کشف سرویس `GET /v1/` است. راه‌اندازی agent از Email OTP روی دامنهٔ auth استفاده می‌کند، یک ApiKey با عمر طولانی برمی‌گرداند، و سپس SQL را از طریق machine API در API Gateway اجرا می‌کند.

برای هر کاربر یک فضای کاری ایزوله ساخته می‌شود. سیاست‌های RLS در هر پرس‌وجو عضویت را بررسی می‌کنند، از جمله در درخواست‌های SQL سمت ماشین.
