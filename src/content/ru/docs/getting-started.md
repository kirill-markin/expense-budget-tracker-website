## Облачная версия

Быстрее всего начать с управляемой облачной версии:

1. Перейдите на [app.expense-budget-tracker.com](https://app.expense-budget-tracker.com)
2. Зарегистрируйтесь по электронной почте: пароль не нужен, вход подтверждается одноразовым кодом
3. Начните учитывать расходы в веб-приложении
4. Если хотите подключить Claude Code, Codex или OpenClaw, передайте агенту `https://api.expense-budget-tracker.com/v1/`

Ничего устанавливать и настраивать на сервере не нужно. Ваши данные изолированы на уровне рабочих пространств благодаря row-level security в Postgres.

## Доступ для агентов и программ

Одна и та же учётная запись подходит для:

- веб-приложения по адресу `https://app.expense-budget-tracker.com`
- нативного подключения агентов через `GET https://api.expense-budget-tracker.com/v1/`
- прямых HTTP-клиентов с `Authorization: ApiKey <key>`

## Самостоятельное размещение

Если вы хотите запускать сервис на своей инфраструктуре, откройте [руководство по самостоятельному размещению](/ru/docs/self-hosting/).

## Демо-режим

В приложении есть встроенный демо-режим. Переключите `All/Demo` в шапке, чтобы посмотреть интерфейс на демонстрационных данных без подключения к базе данных.
