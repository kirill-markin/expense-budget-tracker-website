## Облачная версия

Самый быстрый старт — управляемая облачная версия:

1. Перейдите на [app.expense-budget-tracker.com](https://app.expense-budget-tracker.com)
2. Войдите по email через одноразовый код
3. Начните учитывать расходы в веб-интерфейсе
4. Если хотите подключить Claude Code, Codex или OpenClaw, передайте агенту `https://api.expense-budget-tracker.com/v1/`

Установка и настройка сервера не нужны. Данные изолируются на уровне workspace с помощью Postgres row-level security.

## Доступ для программ и агентов

Один и тот же аккаунт подходит для:

- веб-интерфейса по адресу `https://app.expense-budget-tracker.com`
- агентного онбординга через `GET https://api.expense-budget-tracker.com/v1/`
- прямых HTTP-клиентов с `Authorization: ApiKey <key>`

## Self-hosted

Если вы хотите запускать систему у себя, откройте [руководство по self-hosting](/ru/docs/self-hosting/).

## Демо-режим

В приложении есть встроенный демо-режим. Переключатель All/Demo в шапке показывает интерфейс на примерных данных без подключения к базе.
