## نظرة عامة على النظام

```
واجهة المتصفح  -->  تطبيق الويب Next.js  -->  Postgres (RLS)
                            |                    ^
                            v                    |
                    خدمة المصادقة ---------------+
                            ^
                            |
            عملاء آليون عبر API Gateway
                            ^
                            |
               العامل (جالبو أسعار الصرف) ------
```

خمسة مكونات، وقاعدة بيانات واحدة:

1. **web** — تطبيق Next.js يوفّر لوحات الواجهة ومسارات API
2. **auth** — تسجيل الدخول عبر OTP بالبريد الإلكتروني وتهيئة الوكلاء على نطاق المصادقة
3. **sql-api** — ‏AWS Lambda خلف API Gateway للعملاء الآليين
4. **worker** — يجلب أسعار الصرف اليومية من ECB وCBR وNBS
5. **Postgres** — مصدر الحقيقة الوحيد مع أمان على مستوى الصفوف

## نموذج البيانات

- **ledger_entries** — صف واحد لكل حركة على الحساب (دخل، إنفاق، تحويل)
- **budget_lines** — خلايا ميزانية لا تُعدَّل بعد إضافتها مع اعتماد آخر كتابة
- **budget_comments** — ملاحظات لا تُعدَّل بعد إضافتها ومرتبطة بخلايا الميزانية
- **workspace_settings** — عملة التقارير وإعدادات مساحة العمل
- **account_metadata** — بيانات وصفية لكل حساب مثل تصنيف السيولة
- **exchange_rates** — أسعار صرف يومية تُستخدم للتحويل وقت الاستعلام
- **workspaces** / **workspace_members** — عزل متعدد المستأجرين
- **accounts** — عرض مشتق من ledger entries

## تعدد العملات

تُخزَّن جميع المبالغ بعملتها الأصلية. ويحدث التحويل إلى عملة التقارير وقت القراءة عبر عمليات ربط SQL مع `exchange_rates`. لا توجد أي عملية تحويل مسبقة تفقد الدقة.

## المصادقة

يوجد وضعان عبر متغير البيئة `AUTH_MODE`:

- `none` — بدون مصادقة، مع مساحة عمل محلية واحدة
- `cognito` — ‏OTP بلا كلمة مرور عبر البريد الإلكتروني باستخدام AWS Cognito، مع تسجيل مفتوح

بالنسبة إلى العملاء الآليين، تكون نقطة الاكتشاف العامة هي `GET /v1/`. وتستخدم تهيئة الوكيل OTP عبر البريد الإلكتروني على نطاق المصادقة، ثم تعيد ApiKey طويل العمر، وبعد ذلك تُنفَّذ استعلامات SQL عبر واجهة API الخاصة بالآلات خلف API Gateway.

يحصل كل مستخدم على مساحة عمل معزولة. وتتحقق سياسات RLS من العضوية في كل استعلام، بما في ذلك طلبات SQL الموجهة إلى الآلات.
