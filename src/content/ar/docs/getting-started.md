## النسخة السحابية

أسرع طريقة للبدء هي استخدام النسخة السحابية المُدارة:

1. انتقل إلى [app.expense-budget-tracker.com](https://app.expense-budget-tracker.com)
2. أنشئ حسابًا باستخدام بريدك الإلكتروني عبر رمز OTP بدون كلمة مرور
3. ابدأ بتتبّع المصروفات من تطبيق الويب
4. إذا أردت ربط Claude Code أو Codex أو OpenClaw، فمرّر إلى الوكيل `https://api.expense-budget-tracker.com/v1/`

لا حاجة إلى تثبيت أو إعداد خادم. بياناتك معزولة عبر أمان على مستوى الصفوف لكل مساحة عمل في Postgres.

## الوصول عبر الوكلاء والبرامج

يمكن استخدام الحساب المستضاف نفسه مع:

- واجهة الويب على `https://app.expense-budget-tracker.com`
- واجهة البدء المخصّصة للوكلاء عبر `GET https://api.expense-budget-tracker.com/v1/`
- عملاء HTTP المباشرين باستخدام `Authorization: ApiKey <key>`

## الاستضافة الذاتية

إذا كنت تفضّل تشغيل مثيلك الخاص، فراجع [دليل الاستضافة الذاتية](/docs/self-hosting/).

## وضع العرض التجريبي

يتضمّن التطبيق وضع عرض تجريبي مدمجًا. بدّل زر All/Demo في رأس الصفحة لاستكشاف الواجهة باستخدام بيانات نموذجية، من دون الحاجة إلى قاعدة بيانات.
