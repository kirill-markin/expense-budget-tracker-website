## Хмарна версія

Найшвидший спосіб почати роботу — скористатися керованою хмарною версією:

1. Відкрийте [app.expense-budget-tracker.com](https://app.expense-budget-tracker.com)
2. Зареєструйтеся за допомогою email (одноразовий код без пароля)
3. Почніть відстежувати витрати у вебзастосунку
4. Якщо хочете підключити Claude Code, Codex або OpenClaw, передайте агенту `https://api.expense-budget-tracker.com/v1/`

Без встановлення і без налаштування сервера. Ваші дані ізольовані завдяки row-level security у Postgres на рівні робочого простору.

## Доступ для програм і агентів

Той самий акаунт підходить для:

- вебінтерфейсу за адресою `https://app.expense-budget-tracker.com`
- агентного онбордингу через `GET https://api.expense-budget-tracker.com/v1/`
- прямих HTTP-клієнтів з `Authorization: ApiKey <key>`

## Самостійне розгортання

Якщо ви хочете запускати власний екземпляр, дивіться [посібник із самостійного розгортання](/uk/docs/self-hosting/).

## Демо-режим

У застосунку є вбудований демо-режим. Перемкніть кнопку All/Demo у хедері, щоб переглянути інтерфейс на прикладі тестових даних без підключення бази даних.
