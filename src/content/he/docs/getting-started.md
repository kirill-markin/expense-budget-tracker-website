## גרסת הענן

הדרך המהירה ביותר להתחיל היא להשתמש בגרסת הענן המנוהלת:

1. עברו אל [app.expense-budget-tracker.com](https://app.expense-budget-tracker.com)
2. הירשמו עם כתובת האימייל שלכם באמצעות קוד חד-פעמי ללא סיסמה
3. התחילו לעקוב אחר הוצאות בממשק האינטרנט
4. אם אתם רוצים לחבר את Claude Code, Codex או OpenClaw, תנו לסוכן את `https://api.expense-budget-tracker.com/v1/`

אין צורך בהתקנה או בהגדרת שרת. הנתונים שלכם מבודדים באמצעות row-level security ברמת סביבת העבודה ב-Postgres.

## גישה לסוכנים ולתוכנות

אותו חשבון בענן משמש גם עבור:

- ממשק האינטרנט ב-`https://app.expense-budget-tracker.com`
- תהליך ההתחברות המותאם לסוכנים ב-`GET https://api.expense-budget-tracker.com/v1/`
- לקוחות HTTP ישירים המשתמשים ב-`Authorization: ApiKey <key>`

## אירוח עצמי

אם אתם מעדיפים להפעיל מופע משלכם, עיינו ב-[מדריך לאירוח עצמי](/docs/self-hosting/).

האפליקציה כוללת מצב הדגמה מובנה. השתמשו במתג `All/Demo` בכותרת העליונה כדי לסייר בממשק עם נתוני דוגמה, בלי צורך במסד נתונים.
