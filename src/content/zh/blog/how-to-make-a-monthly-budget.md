一个月税后到手 $4,600，看起来还算宽裕。可等你列完账单，想起每年一次的保险续费，再发现平常买了三次菜就已经超过预估，这份宽裕很快就不见了。学会**如何制定月度预算**，就是趁这个月还没忙起来，先给这些数字各自安排好位置。

你不必做到每一笔钱都花得完美，也不需要一张复杂的电子表格。一份实用的计划，从大概率会到账的钱和真实记录中的支出开始，月底再做一次简短复盘。

下文示例均为假设案例。美元金额只是为了让计算更容易理解；同样的方法适用于任何货币。预算是一种规划工具，本文只提供一般预算知识，不构成财务、投资、税务或法律建议。

## 如何制定月度预算：七步路线图

[Consumer.gov 将预算定义为一份书面月度计划](https://consumer.gov/your-money/making-budget)，并建议采用一个实用循环：月初制定计划，期间记录支出，月底比较结果，再根据复盘调整下个月的计划。

创建月度预算可以分成七步：

1. 计算可用于预算的税后到手收入。
2. 还原真实的经常性支出。
3. 把不规则支出换算成月度分摊额。
4. 选择分类，并为每一类分配计划金额。
5. 检查整份计划是否收支平衡。
6. 记录实际收入和支出，不要重复计算转账。
7. 复盘当月结果，修正最有依据的一项预算假设。

顺序很重要。如果还没弄清收入和真实支出就先定分类限额，表面上也许很整齐，但一次再平常不过的意外开销就可能让它失效。

## 第 1 步：从可用于预算的税后到手收入开始

写下预算月份里你有合理把握会进入账户的收入。以扣完工资代扣项后的到手金额为准。税前工资无法说明，扣除税费和福利计划扣款后，真正还有多少钱可以分配。

**可用于预算的月度收入 = 可靠的税后到手工资 + 预计本月到账的其他可靠收入**

工资、福利金、抚养费或其他收入，只有金额和到账时间都可靠时才纳入预算。一次性进账要单独标注，避免抬高下个月的收入基线。商家退款通常冲减原支出；退税或其他一次性返还款则属于一次性进账。

自己名下账户之间的转账和借款都可能改变账户余额，信用卡额度只说明你最多能借多少。它们都不是收入。

如果你每月只领一次工资，就要列清楚这笔到账资金必须覆盖的全部支出。[2026 年每月发薪怎么做预算](/zh/blog/how-to-budget-monthly-paychecks/)会更深入说明收支时点。

如果收入逐月变化，第一版预算应采用保守且有记录支持的金额。不要把尚未到账的发票款当成已经收到，写进计划。[2026 年如何为不固定收入做预算](/zh/blog/how-to-budget-with-irregular-income/)会完整介绍可变收入基线和高收入月份的处理方法。

## 第 2 步：还原真实月度支出

调出近期的活期账户、信用卡和现金记录，再还原日常生活通常需要多少钱。第一版先如实反映当前支出，再决定哪些地方要改。

美国消费者金融保护局建议，在判断月度预算是否现实之前，[查看几个月的账户和信用卡记录](https://www.consumerfinance.gov/owning-a-home/prepare/assess-your-spending/)，并检查发生频率较低的支出。

这一步需要收集：

- 每月固定义务，例如住房、订阅、保险保费和必须支付的债务还款
- 金额会波动的刚需支出，例如水电、买菜、交通、药品和家庭用品
- 弹性选择，例如外食、娱乐和个人消费
- 容易漏记的支出，例如现金、次要账户或信用卡消费

对于相对稳定的可变支出分类，可以使用多个月的平均值。如果某个分类经常大幅波动，就把计划金额设为记录中一个正常但支出偏高月份的水平。

接下来计算经常性支出的基线：

**每月经常性支出 = 每月固定支出 + 符合实际的可变支出**

想更准确地还原支出，可以参考[2026 年如何计算你的真实月度支出](/zh/blog/how-to-calculate-your-true-monthly-expenses/)。下一步会加入非月度支出，让这条基线更完整。

## 第 3 步：把不规则支出换算成月度分摊额

一年缴一次的账单也属于月度预算。付款并非每月发生，但这笔支出往往可以预见。

使用下面的公式：

**年度支出的月度分摊额 = 年度金额 ÷ 12**

**季度支出的月度分摊额 = 季度金额 ÷ 3**

**每月补足额 = 尚需准备的金额 ÷ 付款前的剩余月数**

| 未来支出 | 金额与时间 | 月度分摊额 |
| --- | ---: | ---: |
| 保险续费 | 每年 $1,200 | $100 |
| 年度订阅 | 每年 $360 | $30 |
| 维修预留金 | 每年 $960 | $80 |
| 6 个月后应付的已知费用 | 尚需准备 $600 | $100 |

列出保险续费、登记费、维护费、教育费用、礼物、医疗支出、年度会员费，以及其他大概率还会发生的支出。尽量根据账单或以往交易来估算。

月度分摊额只是预算中预留的金额。你可以把钱留在原账户，也可以转到自己名下的另一个账户。发生转账时，就按转账记录。账单真正支付时只记一次支出；预算中的预留本身不是第二笔支出。[2026 年如何追踪专项预留金](/zh/blog/how-to-track-sinking-funds/)会更详细地说明这类储备。

## 第 4 步：选择月度预算分类并设定计划金额

好的分类应该帮助你做决定，没必要细到描述每一家商店或收据上的每一件商品。

新手月度预算可以从下面几组开始：

| 分类组 | 需要回答的问题 |
| --- | --- |
| 收入 | 会有哪些钱到账？ |
| 住房和公用事业费 | 维持住所正常运转要花多少钱？ |
| 食品和家庭用品 | 平常每周哪些支出会变化？ |
| 交通 | 工作和日常出行要花多少钱？ |
| 医疗和保险 | 医疗与必要的风险保障要花多少钱？ |
| 必须支付的债务还款 | 这个月必须还多少钱？ |
| 个人和弹性支出 | 计划吃紧时，哪些支出可以调整？ |
| 不规则支出预留 | 哪些未来账单需要每月预留一部分？ |
| 储蓄和其他目标 | 你决定为哪些目标投入资金？ |

如果分别列出数字会影响你的决定，就把这一组拆开。比如买菜基本符合计划，但外食总是超过预算，这两项就应该分成两行。

没有一个适用于所有人的分类比例。先根据实际必付项目、优先事项和交易记录设定金额，再判断剩余收入还能支持哪些安排。[如何用支出分类管理个人预算](/zh/blog/how-to-manage-personal-budget-with-expense-categories/)提供了更完整的设置方法。

## 第 5 步：让月度预算计划收支平衡

把所有计划支出、不规则支出预留和目标分配加起来，再与可用于预算的收入比较。

**计划分配总额 = 经常性支出 + 不规则支出预留 + 储蓄和目标分配**

**计划结余 = 可用于预算的收入 − 计划分配总额**

计划结余为负，说明当前计划安排的金额超过了这个月能提供的钱。先核对各项数字。如果数字无误，就优先保障核心义务和必还款，再依次检查弹性支出、目标期限和金额较大的结构性成本。只有实际到期日允许，延长目标期限才有用。

计划结余为正也需要明确安排。你可以留一部分作为应对日常波动的运营缓冲，也可以增加某个目标的分配额，或留给已知的未来支出。计划不必刚好归零。估算还比较粗略时，保留一笔适度且尚未分配的缓冲会很有用。

如果光是必付义务就已经超过可靠收入，重新调整分类也无法补上全部缺口。预算会把差额清楚显示出来，让你把注意力放在具体支出、收入、付款安排，或当地合适的专业支持上。

## 第 6 步：追踪实际支出，不要重复计算转账

在当月实际发生收入和支出时及时记录，并按分类比较计划金额和实际总额。每周检查一次，通常就能在还有调整余地时发现某一分类正在偏离计划。

把资金移动和新增支出分开：

| 事件 | 预算处理方式 |
| --- | --- |
| 用信用卡购买食品和日用品 | 只记一次食品和日用品支出 |
| 之后从活期账户偿还这张信用卡 | 记作转账或信用卡结算，不再记一次食品和日用品支出 |
| 从活期账户转入自己的储蓄账户 | 记作转账；目标分配另行追踪 |
| 在两个自有账户之间转钱 | 记作转账 |
| 银行手续费 | 记作支出 |
| 从账户取现，转入已纳入记账的现金钱包 | 现金真正花掉前记作转账 |

这样，一笔 $90 的信用卡消费，就不会在还款入账后看起来像花了 $180。

记录过程中，要随时核对账面滚动余额和真实账户余额。交易缺失或重复导入，可能让分类对比看起来很精确，底层总账却是错的。如果总额对不上，可以参考[2026 年如何让预算和银行余额对上](/zh/blog/how-to-reconcile-your-budget-with-your-bank-balance/)。

## 第 7 步：复盘当月结果，修正最有依据的一项预算假设

等交易全部入账后，再完成月末结账：

1. 确认实际收入。
2. 核对账户和信用卡余额。
3. 修正缺失、重复或分类错误的记录。
4. 按分类比较计划金额和实际金额。
5. 解释最大的差额。
6. 判断每项差额是偶发情况，还是可能再次出现。
7. 修正下个月最有依据的一项预算假设。

先做最明确、最有用的调整。买菜连续三次超出计划后，可以提高这一分类的金额；取消订阅后，可以下调订阅预算；收到续费通知后，可以增加不规则支出的月度分摊额。其他已经确定的事实也可以一并更新，但不要因为一个异常月份就重做全部分类。

把原因写下来。“买菜：预算从 $520 提高到 $560，因为已经连续三个月超过 $550”，以后会比“少花一点”更容易评估。

需要更完整的月末流程，可以参考[2026 年每月预算复盘怎么做](/zh/blog/how-to-do-a-monthly-budget-review/)。

## 一个贴近实际的月度预算示例

假设玛雅每月税后到手 $4,600。她的交易记录显示，经常性支出为 $3,330。她还想为不规则支出预留 $300，并为储蓄目标投入 $700。

| 月度预算项目 | 计划金额 |
| --- | ---: |
| 住房 | $1,450 |
| 水电 | $240 |
| 食品和家庭用品 | $520 |
| 交通 | $260 |
| 医疗和保险 | $200 |
| 电话和网络 | $110 |
| 必须支付的债务还款 | $250 |
| 个人消费和外食 | $300 |
| **经常性支出小计** | **$3,330** |
| 不规则支出预留 | $300 |
| 储蓄目标分配 | $700 |
| 运营缓冲 | $270 |
| **已分配收入总额** | **$4,600** |

这份计划能够平衡，因为：

**$4,600 − $3,330 − $300 − $700 = $270 运营缓冲**

月底，经常性支出合计 $3,410。买菜比计划高出 $55，水电高出 $15，个人消费高出 $45；交通和医疗合计比计划低 $35。净超支为 $80，因此运营缓冲还剩 $190。

玛雅没有动用为不规则支出预留的 $300，也没有动用分配给储蓄目标的 $700。$45 的弹性支出超额来自一次庆祝活动，而买菜支出已经连续三个月超过 $550。她只修改下个月的一项假设：把买菜预算从 $520 提高到 $560。运营缓冲随之变为 $230；在出现更多证据前，其他计划金额保持不变。

这个月度预算示例之所以有用，是因为实际支出发生变化时，计划并没有失效。运营缓冲吸收了日常波动，复盘则带来了一项有据可查的调整。

## 一份可以重复使用的月度预算清单

当月开始前：

- 列出可靠的税后到手收入和预计到账日期
- 查看近期交易和当前账户余额
- 计算非月度支出的月度分摊额
- 设定分类金额、目标和运营缓冲
- 确认计划分配总额没有超过收入

当月期间：

- 每笔支出只记录一次
- 正确标注自有账户之间的转账
- 每周比较弹性支出分类的实际金额和计划金额
- 导入或录入交易后核对余额

月底：

- 确认所有已入账的收入和支出
- 比较计划金额、实际金额和差额
- 用直白的语言解释最大偏差
- 为下一份预算修正最有依据的一项预算假设

如果这个月已经开始，就从今天的余额和接下来还要履行的义务入手。不必假装整个月重新开始。[2026 年如何在月中开始做预算](/zh/blog/how-to-start-a-budget-in-the-middle-of-the-month/)会说明完整设置方法。

## 常见问题

### 新手应该使用多少个月度预算分类？

先从上面的九组开始，只拆分那些会掩盖重要决定的分类。第一版预算可以有 8 到 15 个分类，但没有硬性数量要求。

### 储蓄应该算作支出吗？

给储蓄设定计划分配额，避免这笔钱又被安排到其他用途。资金在自己名下的账户之间移动时，应记作转账，而不是支出。按最终仍专门留给该目标的净额来衡量进度。

### 如果某个分类的实际支出超出预算怎么办？

先检查整份计划，再决定怎么处理。其他分类或运营缓冲也许能覆盖差额。记录真实金额，不要用转账分类掩盖这笔支出；到月底再判断这是一次性事件，还是计划金额过低的依据。

### 必须从每月第一天开始吗？

不必。从今天的余额开始，记录已经发生的收入和支出，只为接下来还要支付的义务和支出安排资金。可以把本月当成较短的设置月，为下个月更完整的计划做准备。

### 应该多久检查一次月度预算？

当月开始前制定计划，每周检查一次弹性支出分类，月底核对账目。发生大额消费、薪资变化或导入银行对账单后，可以提前检查。这样的节奏既能发现错误和偏差，也不会让做预算变成每天都要处理的项目。

## 让下个月比这个月更准确

要回答**如何制定月度预算**，真正实用的是一套可以反复执行的循环：从可用于预算的收入开始，根据记录还原支出，为不规则支出按月预留，设定分类金额，让计划收支平衡，记录实际收入和支出但不把转账重复算进去，再修正最有依据的一项预算假设。

你的第一份月度预算难免包含估算，这很正常。把计算过程写清楚，不要把转账算进支出。每完成一个月的预算，就用结果让下个月的计划再准确一点。
